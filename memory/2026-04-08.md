## aitest 项目改造

### 任务背景
改造 aitest 项目以适配 ai-lib-rust v0.9.x 新版运行时 API。

### 分析结果

**ai-lib-rust v0.9.x API 变化：**
1. **AiClient 结构**：`model_id` 字段变为私有，需通过 `manifest.id` 访问
2. **新增 execute_stream_with_cancel_and_stats()**：返回流 + CancelHandle + CallStats
3. **CallStats 字段变化**：移除 `request_id`，新增 `model`, `operation`, `endpoint` 等
4. **AiClientBuilder**：支持 `max_inflight()`, `base_url_override()`, `strict_streaming()`
5. **signals() 方法**：获取运行时信号（如 inflight 请求数）

**兼容性：**
- aitest 现有代码与新版 API 基本兼容
- 主要改动：`Cargo.toml` 依赖改为本地路径引用

### 测试策略

1. **先写测试，看到失败**
   - 创建 `tests/integration_test.rs`
   - 测试覆盖：client 创建、builder API、流式 API、非流式执行、共享客户端
   - 无 API key 时跳过，有 key 时运行完整测试

2. **修复测试**
   - 修复 `model_id` 私有字段访问问题
   - 修复 `CallStats.request_id` 字段不存在问题
   - 添加条件跳过逻辑（无 API key 时）
